# Lógica de Programação com JavaScript

Repositório dedicado ao estudo de lógica de programação utilizando
JavaScript puro, sem frameworks.

## Conteúdos
- Variáveis e tipos
- Condicionais
- Laços de repetição
- Funções
- Resolução de problemas

## Objetivo
Fortalecer a base lógica antes de avançar para React e desenvolvimento backend.

## Observação
Os exercícios evoluem em dificuldade e fazem parte do meu processo de aprendizado.
